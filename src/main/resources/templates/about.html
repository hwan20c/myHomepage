About

안녕하세요.
홈페이지를 찾아와 주셔서 감사합니다.
저는 컴퓨터를 좋아하는 개발자 입니다.
새로운 도전을 즐기며, 재미있고 열정적인 사람들과 일하고 싶습니다.
결과를 도출해 내는 것을 좋아하고, 합의가 안 될시에는 팀 전체가 원하는 방향으로 토의를 하여 해결합니다.
함께하면 즐거운 개발자가 되고 싶습니다.
감사합니다.

language

JAVA와 JS를 할 수 있습니다.

Front-End

javafx로는 java application을 만든 경험이 있고,
bootstrap과 jquery는 현재 홈페이지에서도 사용되고 있습니다.

Back-End

spring과 springboot의 환경에서 개발해 봤으며, gradle을 이용해서 빌드를 하였습니다.
ORM은 mybatis와 JPA(Hibernate)를 사용하였고, login은 spring security를 이용해서 구현 하였습니다.
rest api를 이용해서 데이터를 주고받았고, 필요시에는 ajax를 이용해서 비동기 호출도 사용하였습니다.

Database

DB는 개발 과정때는 mysql을 주로 썼으며, mssql은 실제 데이터가 있는 DB로 사용되었습니다.
mongodb는 개발중에 데이터가 많아서 나중을 위해 mongodb를 사용하였습니다.
flyway를 사용해서, DB 관리(query를 이용해 table 생성, 데이터 crud)를 하였습니다.

Cloud Service

다니던 회사가 민감정보를 관리하는 회사였기 때문에,
ncloud의 민감정보를 저장할 수 있는 공간이 있어서 ncloud를 사용하였고,
개발과정중에 aws는 test cloud로 사용했으며, 현재 사이트도 aws에 올라가 있습니다.
사용된 aws 기술은 https를 적용하기 위해서 ssl을 발급 받았고, route 53을 이용하여 dns를 설정하고,
loadblaancer를 이용해 해당 도메인이 443(https)포트로 접근 하기 위해 설정 하였습니다.

Etc cowork tools

slack으로는 외부&내부 프로젝트 소통 채널로 사용하였습니다.
